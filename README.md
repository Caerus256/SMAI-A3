# Regional Thali Identifier

Indian food image classifier for SMAI Assignment 3 (Task T2.6). Upload a photo of an Indian dish and the app predicts the dish name, maps it to a region (North/South/East/West/North East), estimates calories, and flags common allergens.

## Setup

```bash
git clone <repo-url>
cd T2.6-Regional-Thali-Identifier
pip install -r requirements.txt
```

Place the `Indian Food Images/` folder (80 subfolders, 50 images each) and `indian_food.csv` either inside this directory or one level up.

## How to run

1. Generate metadata (only needed once):
   ```bash
   python data_audit.py
   ```
2. Train models (produces weights in `model/` and figures in `figures/`):
   Open `training.ipynb` and run all cells.

3. Launch the app:
   ```bash
   streamlit run app.py
   ```

## Project structure

```
app.py                  Streamlit web app
training.ipynb          Training notebook (EDA + all models)
data_audit.py           Generates metadata and nutrition JSON
data/
  food_metadata_80.csv
  nutrition_allergens.json
model/                  Trained weights (git-lfs tracked)
figures/                Plots generated by the notebook
utils/
  data_utils.py         Data loading helpers
  predict.py            Inference helper
report.tex              LaTeX report
requirements.txt
```

## Results

| Model | Dish Top-1 | Dish Top-3 | Region Acc |
|---|---|---|---|
| CLIP ViT-B/32 Linear Probe | 74.8% | 89.7% | 84.0% |
| ViT-B/16 Fine-tuned | 69.5% | 85.8% | 82.2% |
| EfficientNet-B0 | 49.0% | 70.3% | 69.3% |
| Simple CNN (from scratch) | ~10% | - | - |

Full ablation in `data/ablation_results.csv`.
